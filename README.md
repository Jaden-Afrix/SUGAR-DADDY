### Manual Installation
```bash
# Clone the repository
git clone https://github.com/Jaden-Afrix/SUGAR-DADDY.git

# Navigate to project directory
cd SUGAR-DADDY

# Install dependencies
npm install

# Start the bot
npm start

# For production use
pm2 start index.js --name "SugarDaddyBot"
```

### Environment Configuration
Create `.env` file with:
```env
SESSION_KEY="your_secure_key_here"
ADMIN_NUMBER="+1234567890"
API_KEY="your_openai_api_key"
SAFETY_MODE="strict" # strict | moderate | off
THEME="purple" # purple | dark | neon
```

## üíª Command Reference

> Full command list: [`COMMANDS.md`](./COMMANDS.md)

### Owner Commands
```
!block [number] - Block user globally
!unblock [number] - Unblock user
!join [link] - Join group via invite
!leave - Make bot leave current group
!bc [message] - Broadcast to all users
```

### Group Management
```
!antitag [on/off] - Prevent random tagging
!welcome [message] - Set custom welcome
!promote [@user] - Make user admin
!demote [@user] - Remove admin status
!antilink [on/off] - Block group links
```

### Media Tools
```
!sticker [image] - Create sticker
!crop [image] - Crop to square
!enhance [image] - Improve quality
!toaudio [video] - Extract audio
!merge [videos] - Combine videos
```

### Entertainment
```
!trivia - Start quiz game
!blackjack - Play 21 game
!rps [choice] - Rock Paper Scissors
!meme [text] - Generate meme
!quote - Get inspired quote
```
